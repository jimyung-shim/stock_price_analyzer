📘 Pandas 교재 내용 기반 프로젝트 활용 내역 정리

본 문서는 **

$$교재명: 판다스 데이터 분석$$

**의 목차를 기준으로, '뉴스 기사량과 주가 변동성 기반 매매 타이밍 분석 모델 구축' 프로젝트 코드(src/)에서 실제로 활용된 Pandas 기능들을 정리한 것입니다.

✅ Part 1. 데이터 입출력 (Input/Output)

교재에서 다룬 CSV 파일 입출력 기능을 사용하여 주가 데이터와 뉴스 데이터를 로드하고, 가공된 최종 데이터셋을 저장했습니다.

|

| 교재 챕터 | 기능 | 프로젝트 내 활용 예시 |
| 2.1 CSV 파일 읽기 | read_csv() | pd.read_csv("Amazon stock data.csv")로 주가 데이터 로드 |
| 2.2 CSV 파일 저장 | to_csv() | df.to_csv("final_dataset.csv")로 최종 전처리 데이터 저장 |

✅ Part 2. 데이터 선택 및 인덱싱

데이터프레임에서 특정 열을 선택하거나, 날짜를 기준으로 데이터를 조회하기 위해 인덱싱 기법을 적극 활용했습니다.

| 교재 챕터 | 기능 | 프로젝트 내 활용 예시 |
| 1.10 열 선택 | df['column'] | df['Close'] 등 분석에 필요한 주가/뉴스 컬럼 선택 |
| 1.12 열 추가 | df['new'] = ... | df['volatility'] = df['High'] - df['Low'] 파생 변수 생성 |
| 1.14 원소 수정 | df.iloc, df.loc | 특정 조건에 맞는 데이터 값 수정 |
| 1.16 인덱스 설정 | set_index() | df.set_index('date')로 시계열 분석을 위한 인덱스 설정 |
| 1.18 인덱스 초기화 | reset_index() | 정렬 후 인덱스를 0부터 다시 매길 때 사용 |
| 1.20 데이터 정렬 | sort_values() | df.sort_values(by='date')로 과거순 정렬 |

✅ Part 3. 데이터 연산 및 통계 분석

프로젝트의 핵심인 '데이터 분석' 단계에서 기술 통계량 계산과 상관관계 분석을 수행했습니다.

| 교재 챕터 | 기능 | 프로젝트 내 활용 예시 |
| 3.1 데이터 미리보기 | head() | 데이터 로드 직후 df.head()로 구조 확인 |
| 3.3 요약 통계 | describe() | 데이터의 분포 확인 |
| 3.5 평균 계산 | mean() | agg({'sentiment': 'mean'})으로 일별 뉴스 평균 감성 계산 |
| 3.8 표준편차 | std() | 주가 변동성(Volatility)의 표준편차 계산 |
| 3.9 상관계수 | corr() | df.corr()을 사용하여 뉴스 감성과 주가 등락률 간 관계 분석 |

✅ Part 4. 데이터 전처리 (Preprocessing)

수집된 Raw 데이터의 결측치를 처리하고, 서로 다른 두 데이터(주가, 뉴스)를 하나로 합치는 과정에서 사용되었습니다.

| 교재 챕터 | 기능 | 프로젝트 내 활용 예시 |
| 5.1 결측치 확인 | isnull() | df.isnull().sum()으로 데이터 누락 확인 |
| 5.2 결측치 제거 | dropna() | 머신러닝 학습 전 NaN이 포함된 행 제거 |
| 5.3 결측치 치환 | fillna() | 뉴스가 없는 날의 기사 수를 0으로 채움 (fillna(0)) |
| 5.5 자료형 변환 | astype() | 문자열로 된 가격 정보(예: "$100")를 float로 변환 |
| 5.16 문자열 조작 | str.replace() | 가격 데이터의 특수문자('$', ',') 제거 |
| 5.20 데이터 결합 | join() / merge() | stock_df.join(news_df, how='left')로 데이터 병합 |

✅ Part 5. 데이터프레임의 구조 조작 (Advanced)

복잡한 시계열 데이터를 분석하기 쉬운 형태로 그룹화하거나 함수를 적용했습니다.

| 교재 챕터 | 기능 | 프로젝트 내 활용 예시 |
| 6.1 함수 매핑 | apply() | df['title'].apply(calculate_sentiment)로 감성 분석 함수 적용 |
| 6.4 그룹 연산 | groupby() | df.groupby('date')로 같은 날짜의 뉴스 기사들을 그룹화 |
| 6.5 집계 함수 | agg() | 그룹화된 데이터의 개수(count)와 평균(mean)을 한 번에 계산 |

✅ Part 6. 시계열 데이터 (Time Series)

주가 데이터의 핵심인 날짜와 시간을 다루는 기능을 사용했습니다.

| 교재 챕터 | 기능 | 프로젝트 내 활용 예시 |
| 7.1 문자열→날짜 | to_datetime() | pd.to_datetime(df['date'])로 날짜 형식 변환 |
| 7.2 시계열 속성 | .dt 접근자 | df['date'].dt.year로 연도/월 정보 추출 |
| 7.4 시프트 | shift() | df['Close'].shift(-1)을 사용하여 '내일의 주가' 타겟 변수 생성 |

✅ Part 7. 데이터 시각화

분석 결과를 그래프로 표현하여 리포트에 활용했습니다.

| 교재 챕터 | 기능 | 프로젝트 내 활용 예시 |
| 4.1 선 그래프 | plot() | Pandas 내장 plotting 기능으로 주가 추이 시각화 |
| 4.2 막대 그래프 | plot(kind='bar') | 연도별 뉴스 기사량 비교 |
| 4.4 산점도 | plot(kind='scatter') | 뉴스 감성 점수와 수익률 간의 관계 시각화 |
